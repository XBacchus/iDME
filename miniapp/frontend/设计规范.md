# 前端设计规范文档

> 基于 `设计模板.html` 提取的设计系统规范

## 一、设计风格定位

**主题**：Earth Edition - 工业暗色主题
**风格关键词**：现代化、专业、简洁、空间感、Arc 浏览器风格
**适用场景**：工业制造管理系统

## 五、特色组件

### 1. 工艺流程节点
```html
<!-- 圆形节点 + 连接线 -->
<div class="process-line">
  <div class="w-14 h-14 circle-icon bg-blue-500">01</div>
  <span>毛坯制造</span>
```

**节点状态**：
- 当前节点：`bg-blue-500` (蓝色实心)

- 检测节点：`bg-amber-500/20 text-amber-500`
- 完成节点：`bg-emerald-500/20 text-emerald-500`

```css
父节点: border-l-2 border-blue-500
子节点: ml-8 (缩进 32px)
连接线: 1px solid rgba(255, 255, 255, 0.1)

```

### 3. 工序卡片
```css

```

**状态色**：
- 进行中：`border-emerald-500`

- 已完成：`border-blue-500`

## 十、开发建议

### 1. 使用 Tailwind CSS
模板使用了 Tailwind CSS，建议继续使用以保持一致性。

### 2. 组件化开发
将以下元素封装为 Vue 组件：
- `FloatingCard.vue` - 浮动岛屿卡片
- `CircleIcon.vue` - 圆形图标容器
- `ProcessNode.vue` - 工艺流程节点
- `BOMTreeNode.vue` - BOM 树节点
- `OperationCard.vue` - 工序卡片

### 3. 保持设计一致性
- 所有卡片使用 `floating-island` 类
- 所有圆形元素使用 `circle-icon` 类
- 按钮统一使用 `rounded-sm` (2px 圆角)
- 输入框统一使用 `rounded` (4px 圆角)

### 4. 暗色主题适配
- 所有文字默认白色
- 输入框背景使用半透明黑色
- 边框使用 `rgba(255, 255, 255, 0.08)`
- 避免使用纯白背景
